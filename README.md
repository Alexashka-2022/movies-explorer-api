# Дипломный проект Яндекс Практикум (backend)

## Интро
Проект представляет собой сервис для поиска фильмов по каталогу

### Для проекта были использованы 
1. Node.js
2. MongoDB
3. mongoose
4. ESLint
5. Celebrate
6. Winston
7. bcryptjs
8. dotenv
9. express-rate-limit
